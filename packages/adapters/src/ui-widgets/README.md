# UI Widgets Adapter

Provides a small overlay shell for runtime status and callouts.

## Requires

- `ui:host` `HTMLElement` service

## Exposes

- `setStatus(...)`
- `showCallout(...)`
- `hideCallout()`
- `destroy()`
