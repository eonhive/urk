# Storage Adapter

Provides namespaced local and session persistence for URK.

## Requires

- optional `storage:local` and `storage:session` services with `Storage`-compatible backends

## Exposes

- `getItem(...)`
- `setItem(...)`
- `removeItem(...)`
- `listKeys(...)`
- `clear(...)`
