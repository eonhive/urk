# Audio Adapter

Provides browser-native audio transport orchestration for URK.

## Requires

- no required host service

## Exposes

- `registerTrack(...)`
- `preload(...)`
- `play(...)`
- `pause()`
- `stop()`
- `setMuted(...)`
- `setVolume(...)`
- `getSnapshot()`
- `subscribe(...)`
