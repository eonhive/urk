# Three Adapter

Provides a small Three.js scene surface behind a URK capability contract.

## Requires

- `three:host` `HTMLElement` service

## Exposes

- `getHost()`
- `getCanvas()`
- `getScene()`
- `getCamera()`
- `getRenderer()`
- `getSize()`
- `resize()`
- `render()`
- `raycast(...)`
