# Input Adapter

Provides keyboard-first input normalization for URK.

## Requires

- optional `input:target` service with `Window`, `Document`, or `HTMLElement`

## Exposes

- `isPressed(...)`
- `bindKey(...)`
- `subscribe(...)`
- `clear()`
