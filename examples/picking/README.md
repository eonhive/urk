# Picking Proof

Route: `/picking/`

This proof shows how URK coordinates DOM targets, a Three scene surface, keyboard input, storage, and overlay feedback through one shared runtime state model.

## Uses

- `@urk/core`
- `@urk/adapters`:
  - `loading`
  - `pointer`
  - `input`
  - `storage`
  - `three`
  - `ui-widgets`

## Files

- `index.html` - static shell
- `styles.css` - proof styling
- `main.ts` - runtime setup, scene sync, interaction, and storage flow
