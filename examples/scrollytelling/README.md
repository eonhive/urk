# Scrollytelling Proof

Route: `/scrollytelling/`

This proof shows URK coordinating section state, internal scrolling, pointer/input navigation, motion, and overlay messaging without relying on a framework wrapper.

## Uses

- `@urk/core`
- `@urk/adapters`:
  - `loading`
  - `pointer`
  - `input`
  - `ui-widgets`

## Files

- `index.html` - static shell
- `styles.css` - proof styling
- `main.ts` - runtime setup, section rendering, and navigation flow
