# URK Deferred

These items are intentionally deferred until the kernel, contracts, and first proof example are stable.

## Deferred platform scope

- giant framework integration matrix
- no-code runtime builder
- advanced plugin marketplace
- engine-grade simulation systems
- large visual editor ambitions

## Deferred technical scope

- many adapters without usage proof
- deep devtools before runtime stability
- broad routing/navigation abstraction before core usage proves it
- large package sprawl before `@urk/core`, `@urk/adapters`, and `@urk/examples` are stable

## Why these are deferred

URK currently wins through:

- kernel quality
- adapter quality
- controller clarity
- runtime state clarity
- one strong interactive web proof

Everything else should remain subordinate to that.
