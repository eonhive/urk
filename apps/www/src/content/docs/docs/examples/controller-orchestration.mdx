# Controller orchestration

Status: current.

This example shows two controllers using lifecycle hooks and events to coordinate adapter capability, runtime state, and teardown.

Open the runnable example: [Controller Orchestration](/examples/controller-orchestration/).

## What it teaches

- initialize a controller
- start orchestration after boot
- coordinate between controllers through a runtime event
- respond to runtime state changes
- dispose cleanly

## Boundary

The example is consumed by the website from private `@urk/examples` internals. It demonstrates current runtime behavior, but the website example package is still private and unstable.
