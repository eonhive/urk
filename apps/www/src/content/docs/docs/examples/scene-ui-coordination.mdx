# Scene/User Interface coordination

Status: current.

The [Scene/UI Bridge example](/examples/scene-ui-bridge/) shows a scene-like browser surface coordinated with a normal User Interface (UI) overlay through runtime services, adapters, controllers, and events.

## What it teaches

- keep renderer-specific logic out of the kernel
- use adapters for browser capabilities
- use controllers for coordination
- keep overlays synchronized through explicit runtime state and events

## Boundary

URK coordinates runtime behavior between surfaces. It does not own the renderer, scene graph, animation engine, or product UI framework.
