# Minimal standalone runtime

Status: current.

This is the first live public URK example. It demonstrates a small runtime surface with kernel lifecycle, loading adapter behavior, runtime state, and event visibility.

Open the runnable example: [Minimal Runtime](/examples/minimal-runtime/).

## What it teaches

- boot a kernel
- register a loading adapter
- run a focused controller
- render runtime state
- render recent events
- dispose the runtime surface cleanly

## Boundary

The example is consumed by the website from private `@urk/examples` internals. Treat the page behavior as a learning surface, not as a stable package contract.
