# Roadmap

The current roadmap focuses on clarity before expansion.

## Current milestone

- public website shell
- canonical docs information architecture
- initial eight live runtime examples
- package overview pages
- experimental known-example playground shell
- public Project docs synced to root canon

## Next maturity steps

- strengthen adapter and controller docs from working examples
- harden public package references
- harden release readiness checks and publish hygiene
- run deeper accessibility checks before launch claims
- keep publish hygiene strict
- add any future public examples one at a time only after the roadmap names the scope

## Deferred package ideas

`@urk/schema` and `@urk/inspector` are not current package pages. They stay deferred until the repo has real workspaces and stable contracts for them.

## Roadmap boundary

Roadmap items are not final Application Programming Interface (API) promises. Future planned examples and experimental wrappers must stay labeled until they are implemented and validated.
