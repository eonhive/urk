# @urk/examples

`@urk/examples` is private, internal, unstable, and consumed by `apps/www`.

It is not a public npm API.

Status: private internal workspace.

Package page: [@urk/examples](/packages/examples/)

## Install

Do not install this package from consumer projects.

Inside the URK repository, build it as a private workspace package:

```bash
corepack yarn workspace @urk/examples build
```

## Current role

The package provides example metadata and a website mount contract for runtime islands.

The current live catalog includes the public runtime examples under `/examples/`, including `minimal-runtime`, `adapter-registration`, `controller-orchestration`, `runtime-state`, `event-routing`, `scene-ui-bridge`, `pointer-input-overlay`, and `embedded-docs-demo`.

## Internal mount shape

The current internal shape is:

```ts
mount(host: HTMLElement, options?: ExampleMountOptions): ExampleMountResult
```

Do not build external packages against this contract yet.

## Current exports

- `exampleCatalog`
- `getExampleMeta(id)`
- `loadExampleModule(id)`
- internal example metadata and mount types
- current runtime mounts for website routes
