# Event types

Runtime events use `KernelEvent` from `@urk/core`.

```ts
export interface KernelEvent {
  type: string;
  timestamp: number;
  source: string;
  payload?: unknown;
}
```

## Event bus

```ts
const offType = ctx.events.on('runtime:phase-changed', (event) => {
  console.log(event.payload);
});

const offAny = ctx.events.onAny((event) => {
  console.log(event.type);
});
```
