# Adapter contract types

Adapter contracts come from `@urk/core` and are re-exported by `@urk/adapters`.

```ts
export interface AdapterRegistration<TApi> {
  id: string;
  capability: string;
  dependsOn?: string[];
  isSupported?: (ctx: RuntimeContext) => boolean;
  setup(ctx: RuntimeContext): Promise<TApi> | TApi;
  dispose?(ctx: RuntimeContext, api: TApi): Promise<void> | void;
}
```

## Lookup

Adapters can be resolved by id or capability.

```ts
const adapter = ctx.adapters.get('loading');
const required = ctx.adapters.require('loading');
```
