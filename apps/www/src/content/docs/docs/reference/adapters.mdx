# @urk/adapters

`@urk/adapters` provides contract-first runtime capabilities.

The package currently exports loading, audio, input, pointer, storage, three, UI widgets, and adapter contract helpers.

Status: current, publishable, 0.x.

Package page: [@urk/adapters](/packages/adapters/)

## Install

```bash
pnpm add @urk/core @urk/adapters
```

## DOM subpath

Use `@urk/adapters/dom` when you only need DOM-facing adapters.

```ts
import { createLoadingAdapter } from '@urk/adapters/dom';
```

## Root import

```ts
import { createLoadingAdapter, createPointerAdapter } from '@urk/adapters';
```

## Current adapters

- `createLoadingAdapter()`
- `createInputAdapter()`
- `createPointerAdapter()`
- `createStorageAdapter()`
- `createUiWidgetsAdapter()`
- `createAudioAdapter()`
- `createThreeAdapter()`

## Boundary

Adapters expose capabilities. They should not own product orchestration or app-specific policy.
