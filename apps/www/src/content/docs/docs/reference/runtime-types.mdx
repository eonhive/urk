# Runtime types

Runtime types live in `@urk/core`.

Key public types include `RuntimePhase`, `RuntimeSnapshot`, `RuntimeContext`, `KernelConfig`, `FrameInfo`, and `FrameScheduler`.

## Runtime snapshot

```ts
export interface RuntimeSnapshot {
  phase: RuntimePhase;
  reason?: string;
  updatedAt: number;
}
```

## Runtime context

```ts
export interface RuntimeContext {
  id: string;
  state: RuntimeStore;
  adapters: AdapterRegistry;
  controllers: ControllerRegistry;
  services: ServiceRegistry;
  events: EventBus<KernelEvent>;
  scheduler: FrameScheduler;
}
```
