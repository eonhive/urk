# URK Docs

URK is the **runtime kernel for interactive browser experiences**.

These docs focus on the parts URK actually owns:

- lifecycle and bootstrap
- runtime context
- explicit runtime state
- adapter registration and capability lookup
- controller lifecycle and orchestration
- event routing foundations
- scene and User Interface bridge foundations

The public brand system is published separately at [the URK brand board](/brand/). It exists to support the public website and documentation shell, not to redefine the kernel boundary.

URK does **not** try to be:

- a full application framework
- a product shell
- a mini game engine clone
- just a React library
- just a state manager
- a no-code product
- a backend platform
