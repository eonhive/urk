# Use URK inside Astro docs

Astro and Starlight own the documentation shell. URK owns live runtime islands inside that shell.

Markdown with JSX-style components (MDX) can host a component that mounts a runtime on the client.

```mdx
import LiveURKExample from '@/components/urk-runtime/LiveURKExample.astro';

<LiveURKExample example="embedded-docs-demo" />
```

Status: current. The runnable public proof is [Embedded Docs Demo](/examples/embedded-docs-demo/).

## Rules

- docs content must render before the runtime island starts
- runtime island failures must stay local to the island
- `@urk/examples` remains private and unstable
