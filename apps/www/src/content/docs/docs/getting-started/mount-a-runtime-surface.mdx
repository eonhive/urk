# Mount a runtime surface

A runtime surface starts with a browser host element.

The page shell can be static HTML, Astro, React, Next.js, or another framework. URK should only own the live runtime surface inside the host.

```ts
export async function mountRuntime(host: HTMLElement) {
  const kernel = createKernel({
    adapters: [createLoadingAdapter()],
  });

  host.textContent = 'Booting URK runtime';
  await kernel.boot();

  return {
    async teardown() {
      await kernel.shutdown('surface:teardown');
      host.textContent = '';
    },
  };
}
```

## Boundary

The shell owns layout, navigation, Search Engine Optimization (SEO), and static content. URK owns the live runtime behavior inside the mounted surface.
