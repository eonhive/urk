# Create your first runtime

Start with the kernel before adding adapters, controllers, or framework wrappers.

```ts
import { createKernel } from '@urk/core';

const kernel = createKernel();

await kernel.boot();
```

After boot, the kernel has a runtime context, a state snapshot, and an event bus.

```ts
const snapshot = kernel.getState();
const events = kernel.getEventBus();

events.onAny((event) => {
  console.log(event.type, event.source);
});
```

## What this proves

- the kernel can be constructed
- the boot lifecycle can run
- runtime state is readable
- runtime events are observable

## Cleanup

Shutdown should happen when the browser surface unmounts or the proof is reset.

```ts
await kernel.shutdown('surface:dispose');
```
