# Installation

Install the runtime pieces you need first.

```bash
pnpm add @urk/core @urk/adapters
```

Use the package manager your project already uses. The repository itself uses Yarn with Node 22, but consumers do not need the repository workspace setup to use the published packages.

## Current packages

- `@urk/core` provides the kernel, runtime context, state store, registries, event bus, scheduler, and public contract types.
- `@urk/adapters` provides current adapter implementations and adapter contract re-exports.
- `@urk/adapters/dom` provides the DOM-only adapter subpath.

## Integration packages

`@urk/react-urk` and `@urk/next-urk` are integration packages. Treat them as wrappers around the standalone runtime model, not as the source of truth.

## Internal package

`@urk/examples` is private, internal, unstable, and consumed by the public website. Do not use it as a stable npm API.
