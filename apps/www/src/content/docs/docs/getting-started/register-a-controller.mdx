# Register a controller

Controllers coordinate runtime behavior. They should not implement adapter internals or render the whole application.

```ts
import type { ControllerRegistration } from '@urk/core';

const loadingController: ControllerRegistration = {
  id: 'loading-flow',
  start(ctx) {
    const loading = ctx.adapters.require('loading');

    loading.begin([
      { id: 'boot', label: 'Boot runtime' },
      { id: 'ready', label: 'Show surface' },
    ]);

    ctx.state.setPhase('ready', 'loading-flow:ready');
    loading.complete('Runtime ready');
  },
};
```

Register the controller at kernel creation time.

```ts
const kernel = createKernel({
  adapters: [createLoadingAdapter()],
  controllers: [loadingController],
});
```
